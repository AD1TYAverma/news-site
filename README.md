# news-site
A dynamic News Website built with PHP and MySQL where users can read latest news articles by category. Admin can manage posts, categories, and users through the admin panel.

# Features
User-friendly news website
Admin dashboard
Add/Edit/Delete news posts
Category management
User management
Responsive design
Search functionality
Pagination
Latest news section
Image upload support

# Technologies Used
PHP
MySQL
HTML5
CSS3
JavaScript
Bootstrap
